Game Name: Typing Speed Test

General Description:

The game aims to test the player's typing speed and accuracy. It includes different difficulty levels (Easy, Medium, and Hard) where words suitable for each level are presented.

Level:

    How to Play:

    1. Select Level: The player chooses a difficulty level from a dropdown menu.
    2. Start Game: Click on the "Start Game" button to begin the game.
    3. Typing: A random word from the selected level's word list appears, and the player must type it into the input box.
    5. Verification: After typing is completed or the time runs out, the correctness of the typed word is checked.
    6. Score: Points are calculated based on the number of correctly typed words.

    Features:

    • The remaining time is continuously updated during the game.
    • Upcoming words are displayed in a list under the current word.
    • After the game ends, a congratulatory message or end-of-game message is shown based on performance.
    • Daily results are recorded in the browser's local storage.

    User Interface:

    • Title Bar: Displays the game title and features a scrolling horizontal line.
    • Game Area: Includes the main user interface elements such as messages, Start Game button, word input field, upcoming word list, and time and score statistics.
    • End Messages: Shows congratulatory messages upon winning or end-of-game messages when time runs out or an incorrect word is typed.
